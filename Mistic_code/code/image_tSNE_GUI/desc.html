Mistic: Image t-SNE viewer for multiplexed images

Interact with the widgets below to plot either a stack montage for all markers of a single multiplxed image, or multiple multiplexed images based on a subset of markers. There are two canvases: the image tSNE canvas to the right with the multiplexed images and the live canvases showing the tSNE scatter plots based on image metadata. Hover over the tSNE dots to get more information about each image.
